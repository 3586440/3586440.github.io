如何搭建自己的博客（hexo篇）

什么是hexo

​ Hexo是一个快速，简单和强大的博客框架。他无需假设自己的服务器和域名，只要你有自己的github账号，就可以自动帮你生成一个属于你自己的域名 和服务器。

​ 如果你想要有一个属于自己的博客，那么hexo是一个很不错的选择！

如何创建hexo

首先要在你的电脑上 安装node.js 和git

安装node.js

https://nodejs.org/en/

​ 点开超链接 如图 ↓

按照自己的电脑系统选择对应的系统安装node.

node安装完毕之后，安装git

安装git

https://git-scm.com/download/win

点开超链接 如图 ↓

按照自己的电脑系统选择对应的系统安装git

两个都安装完成之后，那么恭喜你 已经完成了50% 接下来我们要安装hexo了

安装hexo

在你的硬盘中(随便一个硬盘)创建一个文件夹，然后shift + 鼠标右键 打开Powershell 窗口

在此输入 $ npm install -g hexo-cli 。

windows用户只需要输入 npm install -g hexo-cli，因为我是个小白 我不知道前面的$符号是做什么用的，据我猜想应该是给 MAC 系统用的吧。
等待下载完成

注意：我这里用的是cnpm的下载方式，如果你的npm下载方式很慢，或者下载不成功，可以去百度一下，下载一个cnpm。

接下来我们要在命令行中输入hexo init <folder> 
注意：这里的<folder> 是你要创建的文件夹的名称 打个比方 在命令行中输入hexo init MyApp  那么他会自动帮你创建一个名字叫做MyApp的文件并加载好里面所有的配置文件。
如图↓  



接下来我们要初始化一下。 在命令行中输入npm install

结束
